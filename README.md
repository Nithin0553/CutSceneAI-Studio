# CutSceneAI Studio

CutSceneAI Studio is a production-oriented monorepo for AI-assisted cinematic storytelling workflows.

## Structure

- backend: API and service layer
- cir: scene description schemas and examples
- agents: specialized AI agent modules
- adapters: engine integrations, starting with Unreal
- shared: cross-platform shared code and fixtures
- tests: acceptance, integration, and fixture suites
- infrastructure: deployment and CI assets

## Getting started

1. Review the backend service in backend/
2. Explore the CIR schema examples in cir/
3. Use docker-compose.yml to start local services
